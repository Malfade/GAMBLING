Fullstack интенсив

Для тех, кто хочет понимать весь стек технологий. Научитесь создавать полноценные приложения от интерфейса до сервера.

Программа курса

  • Основы (3 недели): HTML/CSS/JS + основы backend
  • Фреймворки (4 недели): React + Express/Nest.js
  • Интеграция (3 недели): Связь frontend и backend
  • Реальный проект (4 недели): Полноценное приложение
