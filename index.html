01. Программы обучения

Выберите направление, которое соответствует вашим целям и интересам. Все программы включают практические задания и реальные проекты.

Frontend разработка

Полный курс современной фронтенд разработки с нуля до продвинутого уровня. Научитесь создавать интерактивные пользовательские интерфейсы с использованием последних технологий.

14 недель

Основы веба (2 недели)

HTML5, CSS3, основы JavaScript, работа с DevTools, адаптивный дизайн

Продвинутый JavaScript (3 недели)

ES6+, асинхронное программирование, работа с DOM, паттерны проектирования

Фреймворки (4 недели)

React/Vue, состояние приложения, маршрутизация, работа с API

Инструменты и оптимизация (2 недели)

Webpack, тестирование (Jest, Cypress), производительность, PWA

Реальный проект (3 недели)

Разработка фичи для основного продукта компании в команде

Backend разработка

Комплексное обучение серверной разработке и работе с базами данных. Освойте создание надежных и масштабируемых серверных приложений.

14 недель

Основы backend (2 недели)

HTTP, REST API, основы Node.js/Python/Java (на выбор), архитектура приложений

Работа с базами данных (3 недели)

SQL/NoSQL, ORM, миграции, оптимизация запросов, транзакции

Разработка API (3 недели)

Архитектура, аутентификация (JWT, OAuth), документация (Swagger), GraphQL

Микросервисы и DevOps (3 недели)

Docker, Kubernetes, мониторинг (Prometheus, Grafana), логирование (ELK)

Реальный проект (3 недели)

Разработка микросервиса для основного продукта компании

Fullstack интенсив

Для тех, кто хочет понимать весь стек технологий. Научитесь создавать полноценные приложения от интерфейса до сервера.

14 недель

Основы (3 недели)

HTML/CSS/JS + основы backend, работа с Git, основы командной работы

Фреймворки (4 недели)

React + Express/Nest.js (или аналоги), Redux/MobX, SSR (Next.js/Nuxt.js)

Интеграция (3 недели)

Связь frontend и backend, работа с API, WebSockets, авторизация

Реальный проект (4 недели)

Полноценное приложение от идеи до деплоя на production-сервер

05. Часто задаваемые вопросы

Ответы на самые популярные вопросы о нашей программе

Какие требования к участникам программы?

Для участия в программе необходимо базовое понимание программирования (переменные, условия, циклы, функции) и английского языка на уровне чтения документации. Мы принимаем как начинающих разработчиков, так и тех, кто хочет углубить свои знания в конкретном направлении.

Какой график обучения?

Программа рассчитана на 14 недель интенсивного обучения. Занятия проходят 3 раза в неделю по вечерам (19:00-21:00), а также предусмотрены задания для самостоятельной работы. В среднем студенты тратят 15-20 часов в неделю на обучение.

Есть ли трудоустройство после обучения?

Лучшие выпускники получают предложения о работе в нашей компании или у наших партнеров. Однако мы не гарантируем трудоустройство - все зависит от ваших результатов и уровня подготовки. Мы помогаем с составлением резюме и подготовкой к собеседованиям.

Какие технологии мы используем в обучении?

В frontend: HTML5, CSS3, JavaScript (ES6+), React, TypeScript, Webpack. В backend: Node.js/Express или Python/Django (на выбор), REST API, GraphQL, PostgreSQL/MongoDB, Docker. Также изучаем Git, тестирование (Jest, Cypress), CI/CD.

Как проходит процесс обучения?

Обучение строится по принципу "learning by doing". Каждая тема включает теорию, практические задания и code review от менторов. В конце каждого модуля - мини-проект. Финальный проект - разработка реальной фичи для продукта компании в команде.

Готовы начать обучение?

Присоединяйтесь к сотням студентов, которые уже изменили свою карьеру с помощью наших программ. Выберите подходящий курс и начните свой путь в IT уже сегодня.
