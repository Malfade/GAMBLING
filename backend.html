Backend разработка

Комплексное обучение серверной разработке и работе с базами данных. Освойте создание надежных и масштабируемых серверных приложений.

Программа курса

  • Основы backend (2 недели): HTTP, REST API, основы Node.js
  • Работа с базами данных (3 недели): SQL/NoSQL, ORM
  • Разработка API (3 недели): аутентификация, документация
  • Микросервисы и DevOps (3 недели): Docker, Kubernetes
  • Реальный проект (3 недели): Разработка микросервиса
